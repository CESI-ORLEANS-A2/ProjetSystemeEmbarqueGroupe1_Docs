ArduinoUniqueID

This Library gets the Unique ID / Manufacture Serial Number from the Atmel AVR, SAM, SAMD, STM32, and ESP Microcontroller.

Microcontrollers

ArduinoUniqueID supports the Microcontrollers.

  • Atmel AVR - Not Unique Disclaimer
  • Atmel SAM ARM
  • Atmel SAMD ARM
  • STM32
  • Espressif ESP
  • Teensy
  • Raspberry Pi Pico - RP2040

Installation

  • Install the library by Using the Library Manager
  • OR by Importing the .zip library using either the master or one of the releases ZIP files.

Examples

The library comes with [examples](examples). After installing the library you need to restart the Arduino IDE before they can be found under File > Examples > ArduinoUniqueID.
