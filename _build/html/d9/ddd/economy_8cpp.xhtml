void runEconomyModeStep ( )

Exécute une étape de la boucle principale du mode économique.

Cette fonction est appelée à chaque itération de la boucle principale du programme. Elle permet de gérer le mode économique.

Elle réalise deux actions :

  • Réaliser l'acquisition des données.
  • Si l'acquisition est terminée, on sauvegarde les données dans un fichier.
