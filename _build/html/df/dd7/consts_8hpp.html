◆ LIVE_MODE_SERIAL_OUTPUT

#define LIVE_MODE_SERIAL_OUTPUT   OUTPUT_CSV

Format de sortie utilisé en mode live (OUTPUT_JSON ou OUTPUT_CSV)

Si cette variable est à OUTPUT_JSON, les données seront affichées en JSON. Si cette variable est à OUTPUT_CSV, les données seront affichées en CSV.

Le format de sortie dépend de l'utilisation de la station météo en mode maintance. Ce mode peut permettre de récupérer les données depuis un ordinateur et de les traiter directement. Le format JSON est plus facile à traiter pour un ordinateur, mais prend plus de place en mémoire Flash. Le format CSV est plus difficile à traiter pour un ordinateur, mais prend moins de place.
