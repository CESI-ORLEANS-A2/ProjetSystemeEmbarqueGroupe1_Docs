void runConfigurationModeStep ( )

Exécute une étape de la boucle principale du mode de configuration.

Cette fonction est appelée à chaque itération de la boucle principale du programme. Elle permet de gérer le mode de configuration.

Elle réalise une action :

  • Si l'interpréteur avancé n'est pas activé, elle lit les caractères entrés par l'utilisateur et exécute la commande correspondante.
