Description détaillée

Classe représentant un capteur.

Cette classe contient les informations relatives à un capteur.

Elle permet de faciliter la récupération des données des capteurs en fournissant une interface commune pour tous les capteurs. Elle fourni notamment une méthode d'initialisation et une méthode de mesure.
