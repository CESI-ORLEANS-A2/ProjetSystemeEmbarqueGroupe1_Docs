◆ runMaintenanceModeStep()

void runMaintenanceModeStep ( )

Exécute une étape de la boucle principale du mode maintenance.

Cette fonction est appelée à chaque itération de la boucle principale du programme. Elle permet de gérer le mode maintenance.

Elle réalise trois actions :

  • Si l'interpréteur avancé est activé, vérifie si le mode live est activé. Si c'est le cas, on vérifie si des données sont disponibles sur le port série. Si c'est le cas et que l'utilisateur a entré un retour à la ligne, on quitte le mode live.
  • Réaliser l'acquisition des données.
  • Si l'acquisition est terminée, on affiche les données sur le port série.
